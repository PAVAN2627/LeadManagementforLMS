# Plan 2.2: Data Migration & Seeding

## Objective
Create a script/endpoint to migrate the existing `mockData.ts` into MongoDB. This ensures the dashboard is not empty upon switching to the live backend.

## Context
- src/data/mockData.ts
- REQ-10: Data Migration

## Tasks

<task type="auto">
  <name>Create Seed Script</name>
  <files>scripts/seed.ts</files>
  <action>
    Create a standalone script (or API route `api/seed.ts` for ease of execution in serverless) to:
